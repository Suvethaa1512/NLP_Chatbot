# NLP Chatbot using Hugging Face Transformers

## 📌 Project Overview
This project is a chatbot built using DialoGPT model from Hugging Face.

## 🚀 Features
- Conversational AI chatbot
- Uses pretrained transformer model
- Maintains chat history

## 🛠 Technologies Used
- Python
- Transformers
- PyTorch

## 📊 Output
User can interact with chatbot in real time.
